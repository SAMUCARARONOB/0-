Perguntas Frequentes

Como usar os scripts?
Para usar os scripts, você precisa de um executor compatível. Copie o código do script desejado, abra o executor no Roblox, cole o código e execute.
Os scripts são seguros?
Todos os scripts em nosso site são testados e verificados antes de serem adicionados. No entanto, recomendamos sempre usar antivírus e ter cuidado com scripts de fontes desconhecidas.
Posso ser banido por usar scripts?
O uso de scripts viola os termos de serviço do Roblox. Embora o risco de banimento exista, muitos scripts são projetados para evitar detecção. Use por sua própria conta e risco.
Com que frequência os scripts são atualizados?
Nossa equipe verifica e atualiza os scripts regularmente. Scripts quebrados são removidos ou marcados como desatualizados até que uma correção seja disponibilizada.
